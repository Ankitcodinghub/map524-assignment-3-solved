that the app allows the user to view weather information for their device location.

</li>
<li>You may use the API explorer provided by weather API to learn about possible response format. The following are some of the important fields to be extracted from response object.
temp_c, feelslike_c, wind_kph, wind_dir, humidity, uv, vis_km, and condition
</li>
<li>I encourage you to enhance the UI for professional look and feel in the app.</li>
</ul>
